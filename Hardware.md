# Materiales Usados




## 1. Lista de sensores y actuadores utilizados

### Matriz LED 8x8 tipo 1588BS
- Modelo: 1588BS
- Función: Mostrar las notas del juego desplazándose progresivamente.
- Conexiones: 8 pines para filas + 8 pines para columnas.
![App Screenshot](https://img.drz.lazcdn.com/static/bd/p/f56a63b50df98d33bb52e8e47a18bfdf.jpg_720x720q80.jpg)

### 4 Pulsadores
- Modelo: Pulsador estándar de 12×12 mm o tipo “tact switch”.

- Función: Permitir que el jugador pulse a tiempo según las notas que aparecen.
![App Screenshot](https://images-na.ssl-images-amazon.com/images/I/51dYysS29xL._AC_UL495_SR435,495_.jpg)

### Buzzer pasivo

- Modelo: Buzzer pasivo 5V

- Función: Generar sonidos de feedback o acompañamiento rítmico.
![App Screenshot](https://nalber.com/755-large_default/modulo-zumbador-pasivo-compatible-arduino.jpg)

## 2. Lista de otros componentes

- 26 cables M-M:
    - 16 para la matriz 8x8 (1 para cada fila y columna).
    - 8 para los pulsadores.
    - 2 para el buzzer.

- Placa Arduino UNO (1 unidad)

- Protoboards (2 unidades)

- Cable USB tipo B (alimentación y programación)
